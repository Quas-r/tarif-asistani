YEMEK TARİFİ ASİSTANI

Verdiğiniz malzemelere göre size yemek tarifi öneren bot.

Çalıştırmak için öncelikle proje dizinine ".env" adında bir dosya açıp içine şu şekilde OpenAI API key koymanız gerekiyor:
OPENAI_API_KEY=buraya_api_key_gelecek

Bunun haricinde projedeki paketleri kurulu değilse otomatik kurmak için proje dizinindeyken "pip install -r requirements.txt" komutunu kullanabilirsiniz.

Sonrasında dizin içinde iken "python main.py" komutuyla programı çalıştırabilirsiniz.
